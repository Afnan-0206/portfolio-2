## QUICK-START TEMPLATES BY SCENARIO

### SCENARIO 1: APPLYING FOR FULL-TIME JOB

**Step 1: Research**
- Read job description carefully (3 times minimum)
- Identify 5-7 key requirements
- Note company tech stack and culture

**Step 2: Customize Resume**
1. Add relevant projects to top
2. Reorder skills to match job description
3. Adjust bullet points to emphasize relevant experience
4. Tailor professional summary to role

**Step 3: Craft Cover Letter (If Required)**
```
Dear [Hiring Manager Name / Team],

I'm excited to apply for the [Job Title] position at [Company].

As an AI Engineer with expertise in [specific tech from job], I've 
successfully delivered [relevant project]. This experience directly 
aligns with your need for [specific requirement from job description].

[2-3 relevant project examples showing capability]

I'm particularly drawn to [Company] because [specific detail about company/product].

Looking forward to discussing how I can contribute to your team!

Best regards,
Afnan
```

**Step 4: Submit**
- Use name format: "Afnan_B_R_Resume.pdf"
- Add cover letter as separate document
- Submit via specified channel
- Track application deadline

**Key Metrics to Emphasize:**
- Autonomous DevOps agent (70% incident reduction)
- F1 predictor (85% accuracy achievement)
- FormGenius (production deployment)
- Cloud architecture expertise (Vertex AI, GCP)

### SCENARIO 7: INTERVIEW TECHNICAL QUESTIONS

**Q: How would you approach [technical problem]?**

**Your Framework:**
1. **Clarify:** Ask questions about requirements, constraints, scale
2. **Design:** Sketch high-level approach on whiteboard/paper
3. **Implement:** Write pseudocode or actual code
4. **Optimize:** Discuss time/space complexity, improvements
5. **Test:** Walk through test cases

**Q: Tell us about a challenging project**

**Your Answer Framework:**
1. **Context:** What was the project?
2. **Challenge:** What specific problem did you face?
3. **Approach:** What did you try?
4. **Outcome:** How did you solve it?
5. **Lesson:** What did you learn?

**Example:**
"In FormGenius, we faced scalability issues when users created forms with 1000+ fields. The UI slowed down dramatically.

I investigated and found we were re-rendering the entire form on every keystroke. I implemented several optimizations: component memoization, virtualized list rendering, and lazy loading.

This reduced render time from 5 seconds to 500ms - 10x improvement.

The lesson: Performance issues often have obvious solutions if you measure and profile first."

### SCENARIO 8: SALARY NEGOTIATION

**Know Your Market Rate:**
- AI Engineer: $60K-$150K+/year (depending on level)
- Full-Stack: $50K-$120K+/year
- Entry level: $40K-$70K

**Negotiation Approach:**
1. **Don't say a number first** if possible
2. **Research market rates** for your role/location
3. **State range, not single number** (e.g., $70-85K, not $75K)
4. **Justify with:** Years of experience, skills, achievements
5. **Consider total comp:** Salary + bonus + equity + benefits

**Example:**
"Based on market research for AI Engineers in [location] with my experience level and skills (Vertex AI, full-stack, proven projects), I'm looking for $70-85K base with competitive benefits. I'm flexible on exact number depending on equity and professional development opportunities."
