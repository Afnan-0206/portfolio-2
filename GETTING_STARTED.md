- Node.js 18+

- Modern web browser

### Project Structure
```
portfolio 2/
├── src/
│   ├── app/
│   │   ├── page.tsx         (Main page)
│   │   ├── layout.tsx       (Metadata)
│   │   └── globals.css      (Global styles)
│   └── components/
│       ├── ScrollyCanvas.tsx (Canvas animation)
│       ├── Overlay.tsx       (Text overlays)
│       └── Projects.tsx      (Projects grid)
├── public/
```
